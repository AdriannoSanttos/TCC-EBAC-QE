# API Tests - US-0003 Cupons

Testes automatizados para a funcionalidade de Cupons (US-0003) da EBAC Shop.

## Tecnologias
- Jest
- Supertest

## Estrutura
- /tests - Arquivos de teste
- /schemas - Validação de contrato
- /utils - Funções auxiliares
- /reports - Relatórios de execução

## Como executar
npm install
npm test
